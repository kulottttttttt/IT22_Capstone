## Dynamic Data Tables

### Slot Status History
- **Purpose:** Tracks every status change of parking slots
- **Populated:** When staff updates slot status through UI
- **Fields:** SlotId, OldStatus, NewStatus, ChangedAt, ChangedBy, Notes

### Prediction Snapshots
- **Purpose:** Stores historical prediction data
- **Populated:** Generated dynamically by PredictionService
- **Not required:** Predictions calculated on-the-fly

### Parking Incidents
- **Purpose:** Track maintenance issues and incidents
- **Populated:** When users create incidents through Incident Management module
- **Status:** Currently empty (new system)

### Notifications
- **Purpose:** User notifications for events
- **Populated:** System-generated on events (slot changes, incidents, etc.)
- **Status:** Currently empty

## Seed Data Source

**File:** `backend/src/SmartParking.Infrastructure/Persistence/Seed/ApplicationDbContextSeed.cs`

**Execution:** Automatic on application startup via `Program.cs`

```csharp
// Seeding triggered in Program.cs
using (var scope = app.Services.CreateScope())
{
    var context = scope.ServiceProvider.GetRequiredService<ApplicationDbContext>();
    await ApplicationDbContextSeed.SeedAsync(context);
}
```
